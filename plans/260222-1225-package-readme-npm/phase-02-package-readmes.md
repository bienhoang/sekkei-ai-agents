## Implementation Steps

### A. packages/mcp-server/README.md

Structure:
1. **Badges**: npm version, license, node version
2. **Description**: MCP Server for generating Japanese software specification documents (設計書) following V-model chain
3. **Features**: 8 MCP tools, V-model chain, template system, Excel/PDF export, multi-platform
4. **Installation**: `npm install -g sekkei-mcp-server` or `npx sekkei-mcp-server`
5. **Quick Start**: setup → init → generate first doc (3 steps)
6. **Platform Setup**: Claude Code, Cursor, VS Code/Copilot (condensed from root README)
7. **MCP Tools**: 8-row table (tool name, description) — from root README
8. **MCP Resources**: 5-row table (URI, description) — from root README
9. **Document Types**: 5-row table (type, Japanese name, structure) — from root README
10. **Template Customization**: Override dir, resolution order — from root README
11. **Export**: Excel/PDF, Python requirement — from root README
12. **Environment Variables**: 4-row table — from root README
13. **Development**: build/test/lint commands
14. **Ecosystem**: Links to sekkei-preview + sekkei-skills with brief description
15. **日本語**: Japanese summary (5-10 lines covering what/why/how)
16. **License**: MIT

### B. packages/preview/README.md

Structure:
1. **Badges**: npm version, license, node version
2. **Description**: VitePress-based live preview + WYSIWYG editor for Sekkei specification documents
3. **Features**: VitePress SSG, Milkdown WYSIWYG, sidebar nav from config, Vue 3, live reload
4. **Installation**: `npm install -g sekkei-preview` or `npx sekkei-preview`
5. **Quick Start**: generate docs → run preview → edit in browser
6. **Usage**: CLI command, options, how config is resolved
7. **How It Works**: reads sekkei.config.yaml → generates VitePress config → serves docs
8. **Ecosystem**: "Documents generated by sekkei-mcp-server" + "Generate via sekkei-skills"
9. **日本語**: Japanese summary
10. **License**: MIT

### C. packages/skills/README.md

Structure:
1. **Badges**: npm version, license
2. **Description**: Claude Code skill for generating Japanese specification documents following V-model
3. **Prerequisites**: Claude Code CLI, sekkei-mcp-server configured as MCP server
4. **Installation**: `npx sekkei-skills` (auto-install) or manual `cp` to `~/.claude/skills/`
5. **Sub-Commands**: Table with all 19 commands from SKILL.md
6. **Workflow Example**: Step-by-step from init → export showing the V-model chain
7. **Using with MCP Server**: How skills calls MCP tools, configuration
8. **Using with Preview**: Generate → preview → edit cycle
9. **Ecosystem**: Links to sekkei-mcp-server + sekkei-preview
10. **日本語**: Japanese summary
11. **License**: MIT
